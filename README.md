## 🎮 Minecraft Server Bot

Телеграм-бот для автоматической настройки и сборки готовых Minecraft серверов.

![Project Progress](https://img.shields.io/badge/Progress-100%25-brightgreen?style=for-the-badge)

### Что умеет
- ⚙️ Настройка сервера через меню в телеграме
- 🎯 Выбор версии, загрузчика (Fabric/Forge), RAM и всех параметров
- 📦 Генерация готового архива с сервером
